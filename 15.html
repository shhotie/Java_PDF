What are shallow copy and deep copy in java?  To copy the object's data, we have several methods like deep copy and shallow copy.  Shallow copy - The shallow copy only creates a new reference and points to the same object. Example:-- Rectangle obj2 = obj1;  Deep Copy - In a deep copy, we create a new object and copy the old object value to the new object. Example – • Rectangle obj3 = new Rectangle(); • Obj3.length = obj1.length; • Obj3.breadth = obj1.breadth; 47. Why An Interface Cannot Have Constructor In Java?  Interface cannot have constructor in java, because interface not have any instance member so nothing to construct.  Now the question comes how interface can be inherited without constructor because subclass constructor call super class constructor.  We have two cases here. o First case, interface extends other interface: in this situation there is no issue because as we know that no interface have constructor and that’s why no super class constructor will be called. o Second case, a class implements an interface: in this situation there is no inheritance because class implements interface not extends interface. 48.
