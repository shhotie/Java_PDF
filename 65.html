It may be a candidate key or primary key. 31) What are Aggregate and Scalar functions?  An aggregate function performs operations on a collection of values to return a single scalar value.  Aggregate functions are often used with the GROUP BY and HAVING clauses of the SELECT statement.  Following are the widely used SQL aggregate functions: • AVG() - Calculates the mean of a collection of values. • COUNT() - Counts the total number of records in a specific table or view. • MIN() - Calculates the minimum of a collection of values. • MAX() - Calculates the maximum of a collection of values. • SUM() - Calculates the sum of a collection of values. • FIRST() - Fetches the first element in a collection of values. • LAST() - Fetches the last element in a collection of values. ➢ Note: All aggregate functions described above ignore NULL values except for the COUNT function.
