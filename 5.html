Difference between Overloading & Overriding? Overloading Overriding It is a complier time polymorphism. It is run-time polymorphism. It is static binding. It is dynamic binding. In overloading method name is same but different parameter or signature. In overriding method name parameter must be same. We can overload method and constructor both. We can override method but we can’t override constructor. Method overloading is happen in same class but some time it can be happen parent – child class. Method overriding always happen in parent – child class Constructor overloading happen in same class. we cannot override constructor. We can overload static, private and final method. We can’t override static, private and final method. Access modifier (public) and return type (void) does not matter. Access modifier should be same or greater than access modifier.
