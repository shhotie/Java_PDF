What is IOC in spring?  IOC stands for Inversion of Control.  It is heart of spring framework.  It converts object from tight coupling to loose coupling with the help of Container.  Tight Coupling -> Change in one class forces to change in other (dependent) class.  Loose Coupling -> Change in one class doesn’t forces to change in other (dependent) class.  IOC Container is a framework to manage automatic dependency injection throughout application.  There are two types of IOC 1. Core container 2. J2EE Container
