LinkedHashSet 19) Can you add a null element into a TreeSet or HashSet?  In HashSet, only one null element can be added.  In TreeSet it can’t be added as it makes use of NavigableMap for storing the elements.  This is because the NavigableMap is a subtype of SortedMap that doesn’t allow null keys.
